# claude_md — 작업 지식 로그

작업 중 발견한 코드 구조, 패턴, 제약조건을 날짜/주제별로 기록.
에이전트가 동일 영역 재작업 시 중복 탐색을 줄이기 위한 지식 베이스.

---

## 파일 네이밍

```
LOG_YYYYMMDD_<Topic>.md
```

예시:
- `LOG_20260410_Auth_Module_Structure.md`
- `LOG_20260411_Build_Pipeline_Encoding.md`

---

## 파일 포맷

```markdown
---
date: YYYY-MM-DD
topic: 주제 (한글 가능)
related_files: [파일경로1, 파일경로2]
tags: [태그1, 태그2]
---

# 제목

## 발견 내용

...

## 주의사항

...

## 관련 코드 위치

- `파일경로:라인번호` — 설명
```

---

## 저장 시점

- 새로운 모듈/클래스 구조를 파악한 후
- 예상과 다른 동작/패턴을 발견했을 때
- 향후 동일 영역 작업 시 참조 가치가 있는 정보를 얻었을 때
