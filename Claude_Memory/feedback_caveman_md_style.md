프로젝트 기획/스펙 문서(PROJECT.md, REQUIREMENTS.md, ROADMAP.md 류 — 특정 기획 도구 무관)는 caveman style로 작성.

**Why:** 불필요한 설명 없이 핵심 정보만. 가독성·유지보수 효율.

**How to apply:** 문장 대신 키워드/명사구 중심. 동사는 최소화. 설명 문단 지양.
